# Typescript-backend-api

## 구동 시 필요사항

- 명령어

  > npm install  
  > npm run start

- .env

  NODE_SETTING

  > NODE_ENV= production | development  
  > PORT= number  
  > ACCESS_TOKEN_SECRET= string  

  DATABASE_SETTING

  > DB_HOST= string  
  > DB_NAME= string  
  > DB_USERNAME= string  
  > DB_PASSWORD= string

  PW_ENCRYPTION_SETTING
  
  > PW_ENCRYPTION_SALTSIZE= number  
  > PW_ENCRYPTION_ITERATION= number  
  > PW_ENCRYPTION_SIZE= number
